Culture

EASY.INC で働くということ

🎯
本質だけに集中する
不必要な会議、不必要なプロセス、不必要な複雑さを徹底的に排除します。あなたの時間は、本当に大切なことだけに使われます。
"The ability to simplify means to eliminate the unnecessary so that the necessary may speak." — Hans Hofmann
🛠
作りながら考える
議論より実行。プロトタイプより先に会議室で答えは出ません。動くものを作り、使う人に見せ、学び、改善する。そのサイクルを速く回します。
"Talk doesn't cook rice." — Chinese Proverb
🌱
全員が成長する環境
EASY.INC は学習を義務化しません。ただ、学びたい人には惜しみなく機会とリソースを提供します。あなたの「もっと知りたい」を歓迎します。
"An investment in knowledge pays the best interest." — Benjamin Franklin

ポジション概要

EASY.INCのプロダクト開発の中核を担うフルスタックエンジニアを募集します。ユーザーが「難しい」と感じる瞬間をゼロにするプロダクトを、設計から実装まで一気通貫で担当していただきます。

主な業務

  • Webアプリケーションのフロントエンド・バックエンド開発
  • API設計と実装（REST / GraphQL）
  • パフォーマンス最適化とコードレビュー
  • プロダクトマネージャー・デザイナーとの協業
  • 技術的負債の解消と開発プロセス改善

求めるスキル

  • TypeScript / JavaScript 3年以上の実務経験
  • React または Vue.js でのフロントエンド開発経験
  • Node.js / Python などのバックエンド開発経験
  • RDBMSの設計・運用経験
  • 「使いやすさ」への強いこだわり

歓迎スキル

  • スタートアップ環境での開発経験
  • AWS / GCPなどのクラウドサービス経験
  • 0→1のプロダクト開発経験
